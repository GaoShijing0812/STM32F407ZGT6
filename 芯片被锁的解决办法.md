# 第一步

首先下载STM32CubeProgrammer，这里我们要用到CH340，电源接3v3。

# 第二步

进入ISP下载模式

**1.** **按着BOOT0按键上电 上电后松开BOOT0 即可进入ISP模式**

**2.** **我们上电后 先按着RST按键不松 再按着BOOT0按键 松开RST按键 最后松开BOOT0按键 即可进入ISP烧录模试**

使用工具软件里ATK-ISP 不能使用FlyMCU (因为FlyMCU只支持F1系列的主控芯片)

![1781434113869](C:\Users\21722\AppData\Roaming\Typora\typora-user-images\1781434113869.png)

1. 选择我们的COM口

2. 波特率默认115200即可

3. 选择我们要烧录的HEX文件

4. 勾选上 编程后执行

5. 选择 DTR的低电平复位 RTS高电平进BootLoader

6. 点击解除读保护

# 第三步

在STM32CubeProgrammer中使用UART连接，波特率选择115200。在连接前先进入ISP模式，然后即可连接成功。进入Erasing&programming，全选文件后选择擦除芯片（Full chip erase）即可
